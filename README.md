# 🌤️ Weather App

App del clima que muestra la temperatura, condiciones climáticas y recomienda qué ropa usar según el clima de cualquier ciudad del mundo.

## ✨ Características

- Búsqueda de clima por ciudad
- Recomendación de ropa según temperatura
- Fondo dinámico según las condiciones climáticas
- Footer con links a otros proyectos

## 🛠️ Tecnologías

- **Frontend:** React
- **Backend:** Python + FastAPI
- **API:** OpenWeatherMap

## 🚀 Cómo correr el proyecto localmente

### Backend
```bash
cd backend
venv\Scripts\activate
uvicorn main:app --reload
```

### Frontend
```bash
cd frontend
npm start
```

## 📌 Variables de entorno

Crea un archivo `.env` en la carpeta `backend`: API_KEY=tu_api_key_de_openweathermap
